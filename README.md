# lua-rinha-interpreter

An interpreter for ".rinha" files. Made for "Rinha de Compiladores" using Lua

## Files

✅ fib <br/>
✅ combination <br/>
✅ sum <br/>
✅ print

## Execution

```bash
./build.sh
./run.sh <ast-name>
```

### Execution examples

```
./run.sh print
./run.sh sum
./run.sh combination
./run.sh fib
```
